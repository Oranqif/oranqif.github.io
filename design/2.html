Design

I wanted to create an online service app that contains a database of almost every mobile games. Unlike Play Store, this app will solely focus on hosting mobile games with a dedicated filter that categorized them by popularity, rating, genre and gameplay.

Before doing the low fidelity wireframe, I had to get inspiration for the designs from multiple sources using the moodboard. It had helped tremendously in finding the layout, color scheme, the item card, etcetera. I decided to use it for other design projects as well.

To start with the design, I did the low fidelity wireframe first to get an idea of what the app would look like and to test out various designs. It took trial and error to find the right or the clearest possible layout. But once I did, immediately I started to work on the high fidelity wireframe.

For high fidelity, I started on the home page by creating a navigation bar, a search bar, a game list. All the basics for creating a mobile gaming service. Then for the page mainly dedicated for games, I just copied the game list and made it longer. This process was repeated for other similar looking pages. Next, I created the profile page when you register for the mobile app. I had to add a profile picture, friend list, and favorite game list to populate the profile page.
