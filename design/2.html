Design

I wanted to create an online service app that contains a database of almost every mobile games. Unlike Play Store, this app will solely focus on hosting mobile games with a dedicated filter that categorized them by popularity, rating, genre and gameplay.

Before doing the low fidelity wireframe, I had to get inspiration for the designs from multiple sources using the moodboard. It had helped tremendously in finding the layout, color scheme, the item card, etcetera. I decided to use it for other design projects as well.

To start with the design, I did the low fidelity wireframe first to get an idea of what the app would look like and to test out various designs. It took trial and error to find the right or the clearest possible layout. But once I did, immediately I started to work on the high fidelity wireframe.

For high fidelity, I started on the home page by creating a navigation bar, a search bar, a game list. All the basics for creating a mobile gaming service.

Then, for the page mainly dedicated for games, I just copied the game list and made it longer. This process was repeated for other similar looking pages. These pages are mainly for searching mobile games. You can click on the game to send you to the game detail page.

Next, I created the profile page when you register for the mobile app. I had to add a profile picture, friend list, and favorite game list to populate the profile page. The profile page serve as a way to personalize your experience and have a dedicated friend list for mobile games. I added a neat little interaction where you can change your biography with a single click of a mouse.

Then, for the setting page. I inserted separator and texts that would make sense for a online mobile gaming service. Settings like notification, privacy policy, terms and conditions, all the usual settings. I added an interactive toggle switch to simulate changing a setting.

Next, for the all friend related page, I wanted to simulate adding a friend so that I added an interaction for that. So I did, and it turned out great. It even changes the page where all your friends are at.

Finally, the game detail page. I added an image, tags, description for the players to see what kind of game they are about to play.
