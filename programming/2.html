Workout Buddy

Software: Visual Code Studio

Language: JavaScript (React)

Date: 1/21/2025

Introduction

Do you want to get fit? Do you want to make changes to your life? Do you want to keep track of your progress? Then, Workout Buddy is a website for you! Just fill in the details then click on 'Add Workout' so that you can track your progress. Once you are done, delete the task and feel proud that you have achieved your goal.

About the project

This was a short react project I had created that could track all type of exercises possible. The website was integrated with a document database called MongoDB which was where all the data created was stored inside at. MongoDB is different from SQL because the data was stored in JSON-like documents instead of tables. The images below showed what MongoDB and its data looked like...

The main screen of MongoDB

How the data was stored



This website had a backend that would mostly handle managing database from MongoDB. Then, the frontend would call the code from the backend to grab the data and display it inside the website. Additionally, the backend would also handle the error that arised from not filling in all the required fields.

Conclusion

From this short project, I learnt how to create the backend of the website. Furthermore, now I know how to integrate a database from MongoDB which would greatly aid me in future projects.
