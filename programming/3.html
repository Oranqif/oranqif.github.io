Tasks Tracker Mobile Application

Software: WebStorm & Android Studio

Language: JavaScript (React Native)

Date: 04/12/2024

Introduction

Have you had trouble trying to remember what tasks you were supposed to do? Have you ever mixed up a task with another? Have you wished to simply categorize your tasks? Say no more as this app can track all three categories of tasks to complete in your day-to-day basis.

About the project

This app idea came to me when I thought of how I wanted to categorize my tasks so that I do not lose easily mix up or lose track of them. I got my start by first creating a main page that could access to each category...

  • General
  • Grocery
  • Fitness
Each of the categories are styled with different colours to further differentiate them from one another. The image below is the result...

The home screen of the app



Then, I began creating from the first category 'General'. The 'General' category would have tasks that were too broad to be categorized such as 'Go to the library and read something' or 'Do your chores'. The user could add, edit, delete tasks and a checkbox that communicate to the user which ones have had already been done. Furthermore, there was a button that told you how many tasks you have left undone. The images below showed what the 'General' page looked like...

Conclusion

Through this project, I learnt how to develop a proper albeit rudimentary mobile application using WebStorm and Android Studio. Additionally, I also learnt how to apply style and made it looked more aesthetically pleasing and increased readability for the user. If I were to do this again, I would come up with a more complex mobile application that would greatly benefit the user that needed it.
